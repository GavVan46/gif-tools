About GIF Tools

The story behind the tool

What is GIF Tools?

GIF Tools is a free, no-signup web application that lets you split animated GIFs into individual frames, edit them however you like, and rejoin them back into a new GIF.

Whether you want to remove backgrounds, touch up individual frames, or just inspect what's inside a GIF — this tool makes it simple and free.

Who built this?

GIF Tools was built by Gavin Van Vuuren, a developer who wanted a simple, free tool to edit GIF frames without paying for expensive software or dealing with watermarks.

What started as a personal Python script became a desktop app, and then a fully hosted web tool — all built from scratch.

How does it work?

Upload your GIF and the tool splits every frame into a PNG image, packages them into a ZIP file, and lets you download them instantly. Once you've edited your frames in any image editor, simply re-upload them and the tool stitches them back together into a smooth animated GIF — preserving the original frame timing.

Is it really free?

Yes — completely free. No account required, no watermarks, no hidden fees. GIF Tools is a passion project built to be genuinely useful to anyone who needs it.
