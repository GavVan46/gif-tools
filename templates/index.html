Free Online GIF Splitter & Frame Editor

GIF Tools is a free online tool that lets you split any animated GIF into individual PNG frames, edit them in any image editor, and rejoin them back into a smooth animated GIF. Perfect for removing backgrounds from GIFs, touching up individual frames, or creating custom animations.

How to split a GIF into frames

Simply upload your GIF file and click "Split into Frames". Your browser will download a ZIP file containing every frame as a PNG image, along with a timing file to preserve the original animation speed.

How to rejoin GIF frames

After editing your PNG frames, upload them all together using the "Rejoin GIF" tab. GIF Tools will stitch them back together into an animated GIF, preserving your original frame timing.

Remove backgrounds from GIFs for free

Use GIF Tools to extract frames, remove the background from each frame using any free image editor, then rejoin them into a transparent animated GIF.
